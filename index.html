Thinking Like a Programmer

Define different Methods of thinking like a programmer

There are 5 different types of thinking as follows:

  1. Procedural Thinking- Dividing a primary goal/step into multiple small goals/steps that a coputer can understand and process.
  2. Abstract Thinking- Thinking in terms of abstract and general principles to identify similarity between different things.
  3. Systematic Debugging- Methodically identify the problem and find the root cause of the problem to resolve it.
  4. Technological Empathy- Understanding how the computer interpets thing.
  5. Systems Thinking- Breaking down a broader idea into its component pieces.

What is HTML?

HTML stands for Hyper Text Markup Language and it is made of
-text content:What we see
-markup:What it looks like

Tags and Elements

HTML documents are made of HTML elements. This HTML elements can have attributes, which provide additional information about the element and are always specified in the Start Tag and comes in name/value pairs like name = "value". When writing HTML, we tell browsers the type of each element by using HTML tags. This video explains the distinction well.

Inline vs Block Elements

Most of the HTML elements are defined as Inline or Block element. The major difference between Inline and Block elements is that a Block element forms a invisible block around the content in them and the block can have height and width.

What is the Tree-Like Structure of HTML?

It is the Branching Structure of HTML document. What this means is, we can have a ''Parent Tag", which can have multiple "Child Tags". and each "Child Tag" can have "Multiple Child Tags" and so on...

Approach to start working on building a new website

The approach to start working on building a new website can be summarizex as follows:

  1. Print the mock design you got from your Graphic Designer.
  2. Consider the whole printed Page as One Big BOX.
  3. Then Split the One Big BOX into multiple Smaller Boxes that you think you will use while building the website. The basic Idea here is to move from big to small.
  4. Create a blue print of your webpage and more detailed Contents and Styles can be added later.
Chapter 3:Adding CSS Style to HTML Structure

What is CSS?

CSS stands for Cascading Style Sheets and it is defined as computer language used for adding/improving look (think of it as a decoration inside your house) of a document written in amarkup language like HTML

List the ways in which CSS Styling can be applied to HTML document

There are 3 ways in which CSS Styling can be applied to HTML document:

  1. Inline - using a style attribute in HTML elements. It is useful for applying a Uniue Style to a individual HTML element. It uses the style attribute.
  2. Internal - styling is defined in the Head section of an HTML page, using a Style element:
  3. External - styling are defined in an external CSS file, and then linked to in the Head section of an HTML page

How does CSS helps in avoiding repetition of code?

CSS helps in removing the repetition of code in HTML Document. We can create helper classes by creating a set of abstract classes that can be used over and over on HTML elements. Each class is responsible for doing some specific set of styles which then can be used multiple times in the HTML document, thus reducing the need of writing the same code over and over in a HTML document and making the code reusable and scalable for features that can be added later.

Explain the CSS Box Model and its components

According to W3Schools.com the CSS box model is defined as "essentially a box that wraps around HTML elements, and it consists of: margins, borders, padding, and the actual content".The box model allows us to add a border around elements, and to define space between elements.The image below illustrates the box model:

The Box Model

The components of CSS Box Model are as follows:

  1. Content - The content of the box, where text and images appear.
  2. Padding - Clears an area around the content. The padding is transparent.
  3. Border - A border that goes around the padding and content.
  4. Margin - Clears an area outside the border. The margin is transparent.
